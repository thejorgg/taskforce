# Product

## Register

product

## Users

TaskForce is for developers and operators running AI-assisted coding work from a terminal. They need to see what the automation is doing, where it is blocked, and which stage needs attention without losing command-line control.

## Product Purpose

TaskForce is an AI-assisted software workflow tool for developers, not a military project. It coordinates software task intake, dispatch, implementation, review, and release handoff. Success means an operator can trust the pipeline state at a glance, inspect stage output, and approve or redirect work without reading scattered logs.

## Brand Personality

Operational, exact, terminal-native.

## Anti-references

It should not look like a marketing dashboard, a generic SaaS admin template, or a decorative terminal toy. Avoid large empty cards, vague AI assistant copy, and hidden controls.

## Design Principles

- Surface the pipeline first: Echo, Dispatch, Relay, Scope, and Exfil are always visible.
- Make control explicit: key bindings and approval states are never hidden.
- Prefer dense readable telemetry over decorative chrome.
- Keep operator trust high with plain status language and inspectable logs.
- Preserve terminal habits: keyboard-first, scrollable, and predictable.

## Accessibility & Inclusion

Use strong contrast, non-color status labels, keyboard-only operation, and no required motion.
